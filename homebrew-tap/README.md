# Barracuda Homebrew Tap (Template)

This directory is a template for a separate Homebrew tap repo.

## Setup

1. Create a new repo, e.g. `barracuda/homebrew-tap`.
2. Copy the contents of this folder into that repo root.
3. Update `Formula/barracuda.rb` with the latest version and SHA256 values from your GitHub Release.

## Install (end users)

```bash
brew install barracuda/tap/barracuda
```
